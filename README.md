# Жизнь БРО (Без Розовых Очков)

---

## Описание проекта:

Жизнь БРО - система рационального выбора партнёра.

---

###  Цель и назначение

- Методология основана на концепции принятие решений через взвешенные факты, а не через эмоциональные импульсы.
- Система автоматически определяет потенциальную динамику отношений на основе типов привязанности пользователя и
  кандидата.
- Система не заменяет решение человека - она служит зеркалом, фиксирующим паттерны поведения во времени.

### Ограничения системы

1. Система не является объективным судьёй. Входные данные вводит пользователь - субъективность полностью не устраняется.
2. Тип привязанности кандидата определяется ИИ на основе событий - это предположение, а не клинический диагноз.
3. Детектор искажений снижает, но не устраняет эффект ореола при влюблённости.
4. Веса критериев являются фиксированными по умолчанию, но могут быть скорректированы в будущих версиях.

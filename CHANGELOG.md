## [1.0.0] - 2026-05-07

### Добавлено

#### Backend

- Django 5 + Django REST Framework — полноценный REST API
- PostgreSQL 16 — основная база данных
- 15 моделей: пользователи, кандидаты, события, критерии, Hard Stops, отчёты, тесты
- Аутентификация через Telegram ID (TokenAuthentication)
- Пагинация, фильтрация, сортировка для всех списков
- Кастомный обработчик ошибок с единым форматом ответов
- Health check эндпоинт `/health/`

#### AI

- Интеграция с Groq (llama-3.3-70b-versatile)
- Adapter паттерн для поддержки нескольких AI провайдеров
- Анализ событий по 5 критериям с учётом типа привязанности
- Детектор когнитивных искажений
- Валидация ответов AI через Pydantic схемы

#### Скоринг

- Взвешенная система оценки (0-100 баллов)
- 5 критериев: Доверие (30%), Эмоц. стабильность (25%), Уважение (20%), Открытость (15%), Интеллект (10%)
- Автоматическое перераспределение весов при отключении критериев
- Минимум 3 активных критерия

#### Hard Stops

- 7 системных Hard Stops (нельзя удалить, можно отключить)
- Пользовательские настройки независимы для каждого пользователя
- Возможность предложить новый Hard Stop администратору

#### Telegram бот

- aiogram 3.x с FSM состояниями
- Авторизация через Telegram ID с кэшированием токенов
- Добавление и управление кандидатами
- Добавление событий с AI анализом
- Подтверждение и корректировка оценок ИИ
- Настройки: Hard Stops, критерии, тест привязанности
- Тест привязанности с подтверждением результата

#### Отчёты

- PDF отчёты (ReportLab + DejaVu шрифты для кириллицы)
- Excel отчёты (openpyxl, 3 листа: сводка, события, статусы)
- PNG дашборды (Pillow, с круглым аватаром кандидата)
- Текстовые отчёты (HTML в Telegram с фото)
- Политика хранения: файлы удаляются через N дней (настраивается)
- Безопасные имена файлов: транслитерация + UUID + дата

#### Очереди и расписание

- Celery + Redis для асинхронной генерации отчётов
- django-celery-beat — расписание задач через Django Admin
- Еженедельная автогенерация PDF отчётов
- Ежедневная очистка старых файлов

#### Конфигурация

- django-constance — управление константами через Django Admin без изменения кода
- Настраиваемые лимиты токенов, сроки хранения, брендинг

#### Мониторинг

- Sentry — перехват ошибок Python
- Telegram уведомления о критических ошибках
- Ротация логов (RotatingFileHandler)

#### DevOps

- Docker + docker-compose для всех сервисов
- nginx + Let's Encrypt (SSL)
- GitHub Actions CI/CD:
    - CI: lint (isort + black + flake8) + тесты + покрытие
    - Автослияние dev → main после успешного CI
    - CD: сборка образов + деплой на сервер + Telegram уведомления
- Makefile с удобными командами разработки

#### Тесты

- Unit тесты: пользователи, критерии, кандидаты, события, скоринг
- Integration тесты: модели (каскадные удаления), API эндпоинты
- Фабрики (factory-boy) для тестовых данных
- Покрытие кода (pytest-cov)

#### Лендинг

- HTML/CSS/JS лендинг без внешних фреймворков
- Секции: Hero, Stats, Problem, Features, How it works, Modules, CTA
- Scroll reveal анимации
- Адаптивный дизайн
